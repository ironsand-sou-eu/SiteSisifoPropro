Política de privacidade

Quais dados coletamos e tratamos?

O Sísifo Cronos trata dados de navegação, especialmente dos sistemas judiciais, quando (e somente quando) o usuário aciona manualmente a extensão. São tratados, de forma ampla, os dados do processo, inclusive links contidos no processo (como links para decisões e certidões). Este tratamento é restrito ao processo atualmente aberto, NÃO havendo consulta ao histórico de navegação (nem da navegação em geral, nem danavegação pelos sites dos sistemas judiciais).

Além disso, são tratados os dados da conta Google fornecida pelo usuário quando solicitado.

NÃO se recolhe nem se trata qualquer outro tipo de dados. A extensão não realiza registro da navegação (sequer relativa aos sistemas judiciais).

Para que tratamos os dados?

Os dados de navegação do usuário em geral são tratados exclusivamente quando um site está aberto, para verificar se o site atualmente aberto é um sistema judicial compatível.

Os dados de navegação do usuário específicos dos sistemas judiciais são tratados de acordo com as diretrizes do escritório Ruy Andrade Advogados Associados, sendo utilizados exclusivamente para alimentar suas bases de dados de controle de informações processuais em outros sistemas.

Por fim, os dados de login do Google são usados apenas para verificar se o usuário pertence à corporação e para conceder acesso às planilhas Google que contém as configurações do Sísifo Cronos.

Por quanto tempo os dados são mantidos? Qual é a política de segurança?

A extensão NÃO mantém dados do usuário em bases de dados ou outro tipo de persistência de dados. Os dados de navegação, quer gerais, quer de sistemas judiciais, são utilizados pela extensão apenas no momento do uso, para as finalidades descritas acima. Os dados da navegação em si não são guardados pela extensão, e apenas os dados processuais coletados são enviados para os sistemas do escritório, onde são guardados nas respectivas bases de dados.

A única possibilidade de dados pessoais guardados é a eventual indicação do usuário que realizou o cadastro no texto descritivo de determinada informação processual (embora normalmente isso seja feito com um usuário próprio da extensão).

Como não há persistência de dados, a política de segurança dos dados tratados é a do Google, para o tráfego de informações do login na conta Google, e a do sistema utilizado pelo escritório, para o tráfego de informações para o respectivo sistema, atualmente o Projuris. Em ambos os casos, o tráfego se faz com servidores criptografados, com o protocolo HTTPS. Para a política de segurança dos dados após o cadastro nos referidos sistemas, recomendamos buscar a empresa provedora do serviço.

Os dados são compartilhados com algum terceiro?

Os dados tratados pela extensão são compartilhados exclusivamente com o Google e com o sistema utilizado pelo escritório, nas formas descritas acima.
